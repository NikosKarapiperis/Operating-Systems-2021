❏   Διαχείριση σημάτων 
 
 
 
 Παρατηρήσεις/σχόλια 
 Στην συγκεκριμενη περιπτωση δινοντας ctrl+c(το οποιο ειναι το σημα 2) για τον τερματισμο  του προγραμματος , εμφανιζεται στην οθονη ενα μηνυμα στον χρηστη αν θελει οντως να  τερματισει το προγραμμα.Σε περιπτωση που ο χρηστης δωσει την απαντηση ‘y’ τοτε το  προγραμμα σταματαει και επιστρεφει στον φλοιο bash. 
 Το συγκεκριμενο κομματι της εργασιας το υλοποιησα ως εξης: Μεσω της συναρτησης signal(),  με ορισματα τα σηματα τα οποια θελουμε να διαχειριστουμε(SIGINT,SIGTERM) , και την  συναρτηση με την οποια θα το πετυχουμε.Η συναρτηση signalhandler(int sign) , ειναι η  συναρτηση με την οποια θα μπορεσουμε να κατευθυνουμε τα συγκεκριμενα σηματα οπως εμεις  θελουμε.Με την συναρτηση scanf δινουμε την επιλογη στο χρηστη να πληκτρολογησει ‘y’ σε  περιπτωση που θελει οντως να τερματισει το προγραμμα και αν το κανει , κανουμε exit(sign)  ωστε να τερματισει. 
 Σε περιπτωση που δεν επιθυμει να το τερματισει δινει την επιλογη ‘n’ και με την συναρτηση  signal παιρνοντας ως παραμετρο την SIG_IGN αγνοει τα συγκεκριμενα  σηματα(SIGINT,SIGTERM) και συνεχιζεται κανονικα η εκτελεση του προγραμματος.

 Παρατηρήσεις/σχόλια 
 Η καινουργια διεργασια παιδι, η οποια δημιουργηθηκε μεσω της fork , εκτελει μεσω της  εντολης execvp το input που δινει ο χρηστης το οποιο στην συγκεκριμενη περιπτωση ειναι η  εντολη ls -la , η οποια οπως βλεπουμε στο screenshot τρεχει κανονικα χωρις προβληματα και το  exit code που επιστρεφεται παρατηρουμε πως ειναι 0(Αρα ολα τρεξανε σωστα). Κατα την  διαρκεια αυτης της εκτελεσης , η διεργασια πατερας μεσω της εντολης wait ,περιμενει την  διεργασια παιδι να τερματισει. 
 Στην αντιθετη περιπτωση , οπως βλεπουμε και στον κωδικα η perror θα τυπωνει μηνυμα  σφαλματος για την exec και το exit code θα ειχε την τιμη EXIT_FAILURE. 
 Τελος , παρατηρουμε στο τελευταιο screenshot οτι αν δωσουμε λαθος εντολη η εντολη χωρις τις  καταλληλες παραμετρους , το exit code ισουται με 1 οπως θελαμε εξαρχης και η exec μας  εμφανιζει το σωστο σφαλμα.

 Παρατηρήσεις/σχόλια 
 Υλοποιησα τα ζητουμενα(δηλαδη να εμφανιζει σε ποιες γραμμες του file εμφανιστηκε η λεξη) ,  χωρις ομως να καταφερω να τα συνδυασω καπως με τα νηματα. 
 Η υλοποιηση μου ως εξης ειναι : Ανοιγουμε το file με την fopen με παραμετρο ‘r’ ωστε να  διαβασουμε τα περιεχομενα του αρχειου.Επειτα , μεσω της strlen παιρνουμε το μεγεθος του  string και το αποθηκευουμε σε μια μεταβλητη. 
 Mεσω της συναρτησης fgets διαβαζουμε τα δεδομενα του file fp και τα στελνουμε ουσιαστικα στο  array απο chars(buffer) και στην συνεχεια παιρνουμε και το μεγεθος του buffer μεσω παλι της  strlen. 
 Επειτα , συγκρινουμε το string με την λεξη στην θεση i του buffer και αν ειναι ιδιες τυπωνουμε  την γραμμη στην οποια βρεθηκε το string που αναζητουσαμε.Αυτη η αναζητηση συνεχιζεται εως  οτου το i γινει μεγαλυτερο του n(οπου n το μεγεθος του file). 
 Τελος κραταμε στην μεταβλητη line ποσες γραμμες εχει το αρχειο ωστε να γνωριζουμε στην  συνεχεια ποσα νηματα θα δημιουργησουμε(κατι το οποιο δεν πετυχα να υλοποιησω) και  κλεινουμε το αρχειο.
